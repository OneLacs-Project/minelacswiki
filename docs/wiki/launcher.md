# Лаунчеры <br/> <span style="color: gray;"><sup> Какой лучше выбрать? </sup></span>

<span style="color: red;">**TLauncher**</span> и <span style="color: red;">**KLauncher**</span> сразу пропускай мимо. <br>
Воруют информацию, используют не официальные файлы игры, тонна рекламы и т.д.

<span style="color: red;">**FLauncher**</span> тоже пропускай мимо, с ним ты на у нас сервере должно не поиграешь.

[**Legacy Launcher**](https://llaun.ch/ru) - вот это тема для пиратки. Минимализм и капельку рекламы, которую можно навсегда отключить бесплатно.

Если вы богатый роблокс, для лицензии есть такие опции:  [**MultiMC**](https://multimc.org/#Download) или [**PrismLauncher**](https://prismlauncher.org/download/) (форк MultiMC), [**ModrinthApp**](https://modrinth.com/app) (Поддержка `.mrpack`)

[AstralRinth](https://git.astralium.su/didirus/AstralRinth/) - это версия [**ModrinthApp**](https://modrinth.com/app) для пиратов

- Поддержка лицензии и пиратки

- Автообновление лаунчера 
- Автообновление сборки
- Статистика
- Дополнительные, опциональные моды которые можно включить в настройках
